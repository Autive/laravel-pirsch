# Changelog

## [0.3.0] - 2022-07-02

### Added

- Tracking events.

## [0.2.2] - 2022-07-02

### Changed

- No longer send `cf_connecting_ip`, `x_forwarded_for`, `forwarded` and `x_real_ip` headers.

## [0.2.1] - 2022-06-16

### Removed

- Hostname.

## [0.2.0] - 2022-06-14

### Removed

- Honor DNT.

## [0.1.2] - 2022-06-14

### Added

- Hostname.

## [0.1.1] - 2022-06-14

### Added

- Forwarded header.

### Fixed

- X-Real-IP header.

## [0.1.0] - 2022-06-12

### Added

- TrackPageview middleware.
